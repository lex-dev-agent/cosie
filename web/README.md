# Cosie — Promo Website

Static landing page for the Cosie product.

## Structure
- `index.html` — main landing page
- `style.css` — all styles (mobile-first, responsive)
- `script.js` — email signup handling, smooth scroll, scroll-reveal
- `favicon.svg` — bear face favicon

## Running locally

Open `index.html` in a browser, or serve it:

```bash
npx serve .
# or
python3 -m http.server 8080
```

## Design
- Warm neutral palette (cream, terracotta accents)
- DM Sans + DM Serif Display fonts (Google Fonts)
- Mobile-first responsive
- SVG teddy illustration (no image dependencies)
- No build step — pure HTML/CSS/JS
