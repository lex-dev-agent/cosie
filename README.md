# Cosie

**A teddy that watches over your baby — even when you can't.**

Cosie is a cellular-connected teddy bear that monitors your baby's sleep environment temperature and alerts you via mobile app — anywhere, no Wi-Fi needed.

## Key Features

- **Cellular connectivity** — works anywhere, no Wi-Fi required
- **12-month battery life** — zero maintenance for the full SIDS risk window
- **12 months cellular included** — no ongoing subscriptions during the critical period
- **Safe materials** — designed for the nursery environment

## Project Structure
